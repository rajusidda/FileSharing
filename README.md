# Shareby

The aim of this project  :
- Uploading file
- Downloading file
- Share the file with others 

Tech Stack:
- Spring Boot
- JPA
- Restfull 
- Lombok
- H2 Database
- Spring security
- Maven


Endpoint Exposed Endpoints:
- POST /register
- POST /api/file
- GET /api/file/{id}
- POST /api/file
- GET /api/file
